### 📝 نکات مهم بخش Media در HTML

#### 🖼️ تصویر (`img`)

* `src` → مسیر تصویر
* `alt` → توضیح تصویر برای دسترس‌پذیری و وقتی عکس لود نشد
* همیشه برای عکس `alt` بنویس.

#### 🖼️ شکل و توضیح (`figure` و `figcaption`)

* `figure` → گروه‌بندی مدیا
* `figcaption` → توضیح یا کپشن مدیا
* مناسب تصاویر، نمودارها و کدها.

#### 🎥 ویدئو (`video`)

* `controls` → نمایش دکمه‌های پخش
* `poster` → تصویر قبل از شروع ویدئو
* `autoplay` → پخش خودکار
* `muted` → بی‌صدا کردن ویدئو
* `loop` → تکرار ویدئو

#### 🎥 `source`

* برای ارائه چند فرمت از یک فایل:

```html
<source src="movie.mp4" type="video/mp4">
<source src="movie.webm" type="video/webm">
```

* مرورگر خودش فرمت مناسب را انتخاب می‌کند.

#### 🎬 زیرنویس (`track`)

* اضافه کردن زیرنویس به ویدئو
* فایل زیرنویس باید `.vtt` باشد.
* ویژگی‌های مهم:

  * `kind="subtitles"`
  * `srclang="fa"`
  * `label="فارسی"`

#### 🎵 صوت (`audio`)

* `controls` → نمایش کنترل‌های پخش
* می‌تواند چند `source` داشته باشد.
* از `autoplay` و `loop` هم پشتیبانی می‌کند.

#### 💬 نقل‌قول (`blockquote`)

* برای نقل‌قول‌های طولانی استفاده می‌شود.
* `cite` → منبع نقل‌قول

---

### ⭐ نکات حرفه‌ای

✅ برای `video` و `audio` همیشه متن جایگزین بنویس:

```html
مرورگر شما از پخش ویدئو پشتیبانی نمی‌کند.
```

✅ برای مدیاهای مهم از `figure` و `figcaption` استفاده کن.

✅ همیشه چند فرمت (`mp4`, `webm`, `mp3`, `ogg`) ارائه بده تا در همه مرورگرها اجرا شوند.

✅ `alt` و `track` باعث دسترس‌پذیری (Accessibility) بهتر سایت می‌شوند.

---

### چک‌لیست مرور سریع

* `img` → نمایش تصویر
* `alt` → توضیح تصویر
* `figure` → گروه‌بندی مدیا
* `figcaption` → کپشن مدیا
* `video` → نمایش ویدئو
* `poster` → تصویر قبل از پخش
* `source` → چند فرمت فایل
* `track` → زیرنویس
* `audio` → پخش صوت
* `blockquote` → نقل‌قول + منبع (`cite`)



عالیه 👏 این نسخه نسبت به قبلی تمیزتر و حرفه‌ای‌تر شده. اگر بخوام مثل یک منتور HTML کدت رو بررسی کنم، **۸.۵ از ۱۰** می‌گیره.

### ✅ کارهای خوبی که انجام دادی

* استفاده از `figure` و `figcaption`
* نوشتن `alt` برای همه تصاویر
* استفاده از `poster` در ویدئو
* استفاده از چند `source` برای ویدئو و صوت
* اضافه کردن `track` برای زیرنویس
* استفاده از `blockquote` و `cite`
* جدا کردن بخش‌ها با `section` و `article`

## 🔴 نکته ۱: زبان صفحه

الان نوشتی:

```html
<html lang="en">
```

درحالی‌که محتوا فارسی است:

```html
<html lang="fa">
```

---

## 🔴 نکته ۲: جهت صفحه

برای صفحات فارسی بهتر است:

```html
<html lang="fa" dir="rtl">
```

---

## 🟡 نکته ۳: `blockquote`

`blockquote` برای نقل‌قول‌های چندخطی است. اگر جمله کوتاه باشد بهتر است:

```html
<p>
  <q>جدایی دردناک و وصال دردناک‌تر</q>
</p>
```

البته استفاده فعلی‌ات هم اشتباه محسوب نمی‌شود.

---

## 🟡 نکته ۴: گالری تصاویر

الان سه `figure` را داخل یک `article` گذاشتی. از نظر معنایی بهتر است:

```html
<section>
  <h2>گالری تصاویر</h2>

  <figure>...</figure>
  <figure>...</figure>
  <figure>...</figure>
</section>
```

چون این‌ها مقاله نیستند، بلکه یک گالری هستند.

---

## 🟡 نکته ۵: عنوان برای بخش‌ها

هر `section` بهتر است یک عنوان داشته باشد:

```html
<section>
  <h2>تریلر و موسیقی</h2>
  ...
</section>

<section>
  <h2>گالری تصاویر</h2>
  ...
</section>
```

---

## ⭐ نسخه حرفه‌ای `<video>`

اگر بخواهی استانداردتر بنویسی:

```html
<video controls poster="./images/images (3).jpg" width="700">
  <source src="./videos/movie.mp4" type="video/mp4">
  <source src="./videos/movie.webm" type="video/webm">
  مرورگر شما از ویدئو پشتیبانی نمی‌کند.
</video>
```

# 📝 نکات نوت‌برداری

* `lang="fa"` → زبان فارسی
* `dir="rtl"` → راست‌چین کردن صفحه فارسی
* `img` → نمایش تصویر
* `alt` → توضیح تصویر
* `figure` → گروه‌بندی مدیا
* `figcaption` → کپشن مدیا
* `video controls` → نمایش کنترل‌های پخش
* `poster` → تصویر قبل از شروع ویدئو
* `source` → پشتیبانی از چند فرمت
* `track` → افزودن زیرنویس
* `audio controls` → پخش فایل صوتی
* `blockquote` → نقل‌قول بلند
* `q` → نقل‌قول کوتاه
* هر `section` بهتر است یک `h2` داشته باشد.
* برای گالری تصاویر، `section + figure` مناسب‌تر از `article` است.
