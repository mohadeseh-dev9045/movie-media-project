# 🎬 Movie Media Project

A practice project built with pure HTML5 to learn and implement media elements and semantic tags.

## 🚀 Features

* Semantic HTML structure
* Images with `<figure>` and `<figcaption>`
* Videos with multiple `<source>` formats
* Video poster support
* Audio player with fallback sources
* Subtitles using `<track>`
* Blockquote with `cite`
* Organized media gallery

## 🛠️ Technologies

* HTML5

## 📚 Concepts Practiced

* `<img>`
* `<figure>`
* `<figcaption>`
* `<video>`
* `<audio>`
* `<source>`
* `<track>`
* `<blockquote>`
* Semantic elements (`header`, `section`, `article`)

## 📂 Project Structure

```text
images/
videos/
audio/
subtitles/
index.html
README.md
```

## 🎯 Goal

Practice HTML5 media elements and build a semantic movie showcase page without using CSS or JavaScript.
